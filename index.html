The "Title Case" is an application that takes a sentence input from the user
and converts it into title case format. Title case means that the first letter of each word is capitalized,
while all other letters are in lowercase. The application achieves this by utilizing a simple algorithm
that iterates through each word in the input sentence and capitalizes the first letter of each word.
Overall, the Title Case Converter is a simple yet helpful tool that quickly transforms a regular sentence
into an aesthetically pleasing format often used for titles, headings, and headlines in various contexts.
